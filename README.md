# Library Recommendation System

This repository contains a simple library recommendation system scaffold with separate frontend, backend, and data pipeline directories.

## Structure

- `frontend/` - React application files
- `backend/` - Flask API server and recommendation logic
- `data_pipeline/` - Data preprocessing and model generation scripts

## Getting Started

1. Install backend dependencies:
   ```bash
   pip install -r backend/requirements.txt
   ```
2. Install frontend dependencies and run the React app in `frontend/`.
3. Start the backend with `python backend/app.py`.
